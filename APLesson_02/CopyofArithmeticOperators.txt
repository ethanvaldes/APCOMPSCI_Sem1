Operator
	Description
	+
	Additive operator (adds values on either side of the operator)
	-
	Subtraction operator (subtracts value on right side of operator from value on left side)
	*
	Multiplication operator (multiplies values on either side of operator)
	/
	Division operator (divides value on left side of operator by value on right side)
	%
	Remainder operator (divides value on left side of operator by value on right side and gives the remainder)
